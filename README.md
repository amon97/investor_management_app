💰 配当管理兼ニュース集約Webアプリ (Beta)

- このappはclaudecode,antigravityを使用しています

- シニア投資家（60歳以上）向けの、アクセシビリティ最優先の資産管理ツールです [cite: 2026-02-14]。
- 複雑な操作を排除し、「今、いくら持っていて、いつ配当が入るか」を直感的に把握できます [cite: 2026-02-14]。

🌟 特徴
- シニア向けデザイン: 本文18px以上、ボタン48px以上の巨大設計 [cite: 2026-02-14]。

- 資産の可視化: 総資産と年間配当金をグラフで一目瞭然に表示 [cite: 2026-02-14]。

- 銘柄情報の自動入力: 証券コードだけで銘柄名やセクター（業界）を自動補完 [cite: 2026-02-15]。

- ニュース集約: 保有銘柄に関連するニュースをRSS/APIから自動収集 [cite: 2026-01-22, 2026-02-15]。

- 簡単ログイン: Googleアカウント連携によるパスワード不要の認証 [cite: 2026-02-14]。
